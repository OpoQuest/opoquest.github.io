<h2>1. Responsable del tratamiento</h2>
<p>OpoQuest · opoquest.app@gmail.com</p>

<h2>2. Datos que recogemos</h2>
<p>Recogemos los siguientes datos personales: dirección de correo electrónico, nombre de usuario y datos de progreso de estudio (respuestas, estadísticas, rachas).</p>

<h2>3. Finalidad del tratamiento</h2>
<p>Los datos se usan exclusivamente para: gestionar tu cuenta, guardar tu progreso de estudio y mejorar la experiencia de la app.</p>

<h2>4. Base legal</h2>
<p>El tratamiento se basa en el consentimiento del usuario al registrarse en la app.</p>

<h2>5. Conservación de datos</h2>
<p>Los datos se conservan mientras la cuenta esté activa. Puedes solicitar la eliminación en cualquier momento escribiendo a opoquest.app@gmail.com.</p>

<h2>6. Compartición de datos</h2>
<p>No vendemos ni compartimos tus datos con terceros. Utilizamos Supabase como proveedor de base de datos, que cumple con el RGPD.</p>

<h2>7. Tus derechos</h2>
<p>Tienes derecho a acceder, rectificar y suprimir tus datos. Para ejercerlos, escríbenos a opoquest.app@gmail.com.</p>

<h2>8. Cambios en esta política</h2>
<p>Podemos actualizar esta política ocasionalmente. Te notificaremos cualquier cambio significativo a través de la app.</p>

<h2>9. Contacto</h2>
<p>Para cualquier consulta: opoquest.app@gmail.com</p>
